## 使用说明
1. 将个人简历内容填入`resume.txt`文件
2. 替换`resume.pdf`为实际简历文件
3. 更新`about.html`中的个人信息
4. 用户访问AI问答页面时输入自己的Gemini API Key即可使用AI助手功能

项目现已可直接部署使用！
